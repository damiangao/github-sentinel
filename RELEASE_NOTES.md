# GitHub Sentinel v0.0.1

这是 GitHub Sentinel 的第一个 alpha 版本，提供了基本的项目框架和核心功能实现。

## 🚀 新特性

### 核心功能
- 基础项目结构搭建
- 配置管理系统
- GitHub API 客户端封装
- 数据模型定义
- 通知系统框架

### 命令行工具
- 初始化配置 (`init`)
- 仓库订阅管理 (`subscribe`)
- 监控启动 (`start`)

## 📦 技术栈
- Python 3.8+
- PyGithub
- Pydantic
- Click
- Rich
- aiohttp

## 🔧 配置选项
- GitHub Token 配置
- 仓库订阅管理
- 通知方式设置（邮件/Slack/Webhook）
- 监控频率设置

## 🔍 使用说明

1. 安装：
```bash
pip install github-sentinel
```

2. 初始化配置：
```bash
github-sentinel init
```

3. 订阅仓库：
```bash
github-sentinel subscribe owner/repo
```

4. 启动监控：
```bash
github-sentinel start
```
